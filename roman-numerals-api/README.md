Roman Numerals API
===

Requirements
===
Node: 14.0.0
NPM: 6.14.4

####Troubleshoot
You're going to need to create a .env file in this project's root with this configurations:
```javascript
APP_VERSION=v1
APP_LISTEN_PORT=3600
APP_ENV=local
APP_LOGS_DIR=./logs
APP_DB_PATH=mongodb://mongodb:27017/sovrn
```
